Tier 2: Intermediate — Object-Oriented Thinking
Purpose

This section focuses on developing structured thinking in C#, not just writing code that works.

The goal is to:

Understand how to organize code using object-oriented principles
Recognize patterns used in real-world applications
Write cleaner, more maintainable programs
Move from “getting it to work” → “designing it properly”

At this stage, the emphasis is on:

Understanding relationships between classes
Writing reusable and scalable code
Thinking beyond single-file programs
Concepts Covered
Object-Oriented Programming (Core)
Encapsulation (bundling data and behavior together)
Inheritance (reusing and extending existing classes)
Polymorphism (same interface, different behavior)
Abstraction & Design Basics
Interfaces
Defining contracts for behavior
Abstract classes
Shared base functionality with partial implementation
When to use interfaces vs abstract classes (basic understanding)
Data & Memory Concepts
Reference types vs value types
How data is stored and passed
Understanding how objects behave in memory (basic level)
Null Handling
Identifying and preventing null reference issues
Writing safer code with null checks
Generics
Using generic types (<T>) for reusable code
Understanding why generics are useful
Enums & Structs
Using enums for fixed sets of values
Understanding structs as lightweight data containers
Dependency Injection (Usage)
Understanding the idea of passing dependencies into classes
Using DI without needing to design frameworks
Asynchronous Programming
Using async and await
Understanding non-blocking operations at a basic level
LINQ (Basic Usage)
Filtering data with .Where()
Transforming data with .Select()
Unit Testing Basics
Writing simple unit tests
Understanding expected vs actual results
Verifying code behavior
Expected Outcome

After completing this tier, you should be able to:

Design programs using basic object-oriented principles
Structure code across multiple classes effectively
Use interfaces and inheritance in simple scenarios
Work with collections and transform data using LINQ
Write safer code with proper null handling
Understand and apply basic async programming
Use dependency injection in simple applications
Write basic unit tests to validate functionality
